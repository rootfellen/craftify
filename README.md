## Features

- Create and customize your own store.

- Payment transactions with Stripe.

- Dark theme.

- Responsive UI.

- Keep track of stock and orders.

- Easy Clerk Authentication

- Add / Delete / Customize Products, Categories, Sizes, Colors, Stock, etc..
